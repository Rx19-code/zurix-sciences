🔟 INSTALAR DEPENDÊNCIAS (20 min)

MOZILLA+VPN

Copie e cole cada bloco no terminal do VPS:

Bloco 1 - Atualizar sistema:

apt update && apt upgrade -y

Bloco 2 - Instalar programas:

apt install -y nginx python3 python3-pip nodejs npm git curl ufw fail2ban mongodb-server

Bloco 3 - Instalar PM2 e Yarn:

npm install -g pm2 yarn

Bloco 4 - Iniciar MongoDB:

systemctl start mongodb
systemctl enable mongodb

Bloco 5 - Configurar Firewall:

ufw allow 22
ufw allow 80
ufw allow 443
ufw --force enable

Bloco 6 - Criar diretórios:

mkdir -p /var/www/zurix/backend
mkdir -p /var/www/zurix/frontend
Dependências Instaladas

1️⃣1️⃣ COPIAR ARQUIVOS PARA VPS (10 min)

MOZILLA+VPN
⚠️ Execute estes comandos NO SEU COMPUTADOR, não no VPS!
Abra um NOVO terminal/janela.

Copiar Backend:

scp -r /CAMINHO/PARA/backend/* root@SEU_IP:/var/www/zurix/backend/

Copiar Frontend:

  • /CAMINHO/PARA/ = pasta onde extraiu os arquivos
  • SEU_IP = IP do VPS (do papel)
Arquivos Copiados

1️⃣2️⃣ CONFIGURAR BACKEND (10 min)

MOZILLA+VPN

Volte para o terminal conectado ao VPS:

Bloco 1 - Ir para pasta:

cd /var/www/zurix/backend

Bloco 2 - Criar .env:

⚠️ Substitua SEUDOMINIO.com pelo domínio que você comprou!
cat > .env << 'EOF'
MONGO_URL=mongodb://localhost:27017
DB_NAME=zurix_db
CORS_ORIGINS=https://SEUDOMINIO.com
EOF

Bloco 3 - Instalar dependências:

pip3 install -r requirements.txt

Bloco 4 - Popular banco:

python3 seed_database.py
python3 seed_protocols.py

Bloco 5 - Iniciar backend:

pm2 start "uvicorn server:app --host 0.0.0.0 --port 8001" --name zurix-backend
pm2 save
pm2 startup
Backend Configurado

1️⃣3️⃣ CONFIGURAR FRONTEND (10 min)

MOZILLA+VPN

Bloco 1 - Ir para pasta:

cd /var/www/zurix/frontend

Bloco 2 - Criar .env:

⚠️ Substitua SEUDOMINIO.com!
echo "REACT_APP_BACKEND_URL=https://SEUDOMINIO.com" > .env

Bloco 3 - Instalar e buildar:

Frontend Configurado

1️⃣4️⃣ CONFIGURAR NGINX (10 min)

MOZILLA+VPN
⚠️ Substitua SEUDOMINIO.com nos DOIS lugares!
cat > /etc/nginx/sites-available/zurix << 'EOF'
server {
    listen 80;
    server_name SEUDOMINIO.com www.SEUDOMINIO.com;
    
    location / {
        root /var/www/zurix/frontend/build;
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:8001;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF

Ativar site:

ln -sf /etc/nginx/sites-available/zurix /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t
systemctl restart nginx
Nginx Configurado

1️⃣5️⃣ INSTALAR SSL/HTTPS (10 min)

MOZILLA+VPN

Bloco 1 - Instalar Certbot:

apt install -y certbot python3-certbot-nginx

Bloco 2 - Obter certificado:

⚠️ Substitua SEUDOMINIO.com! Use o email ProtonMail quando pedir.
certbot --nginx -d SEUDOMINIO.com -d www.SEUDOMINIO.com

Quando perguntar sobre redirect, escolha opção 2

Bloco 3 - Reiniciar:

systemctl restart nginx
pm2 restart all
SSL Instalado
